# Portfolio (Scaffold)

This is a simple colourful portfolio scaffold.

Files:
- `index.html` — main page
- `styles.css` — styles
- `script.js` — interactivity (theme toggle)
- `assets/photo.svg` — placeholder image (replace with your photo)
- `pdfs/html5.pdf`, `pdfs/css3.pdf`, `pdfs/js.pdf` — course PDFs (placeholders)

How to preview:
1. Open `index.html` in a browser (double-click or use Live Server extension).
2. Replace `assets/photo.svg` with your real photo (use same filename or update `index.html`).
3. To change course certificate images, open `index.html` and edit the `src` attribute on the `<img class="course-preview">` for each course (for example, `assets/html5.jpg`).
4. Replace the placeholder PDFs in `pdfs/` with your actual course files.

Customize:
- Edit text in `index.html` to add your name, bio, email.
- Tweak colors in `styles.css` using CSS variables at the top.
